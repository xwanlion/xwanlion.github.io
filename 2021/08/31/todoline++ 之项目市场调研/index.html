todoline++ 项目市场调研   2021-08-31


1. 项目简述

目前市场上有很多的待办软件和应用, 而且, 待办软件市场已经趋于成熟, 自己也使用过一些软件, 比如 todoist，滴答清单等, 但最后, 自己都放弃了, 原因是, 这些应用都是需要联网才能使用, 数据都是保存在服务器上, 而这些数据都是及其私密的信息, 把这些私密信息保存在别人的服务器上, 总有一种不放心的感觉, 因此, 自己苦苦寻找一款开源软件, 以减轻自己的后顾之忧, 功夫不负有心人, 终于找到了一款应用, 那就是 supperProductive, 但用不了多久, 自己就放弃了, 原因是, supperProductive 是有时候很耗性能, cpu 飙升, 了解后才知道, 这个软件使用 electron 来开发, electron 开发出来的软件, 性能都是这样, 因为编译时不是直接编译为字节码, 而是编译为 js 代码, 然后套在一个浏览器的外壳中运行, 这种方式肯定耗性能。

当我看到cpu飙升时一点都不爽, 况且, supperProductive 并没有我需要的功能, 比如记录习惯， 冥想功能等等。

于是, 我决定自己开发一个 to-do list 应用。

2. to-do list 应用市场潜力

到目前为止, 全球手机用户 39.5 亿, 接近 40 亿, pc(包括笔记本)用户 22.5 亿, 总共 62 亿台设备。

在疫情后复苏的背景下, 不管是pc还是手机, 销量都已经出现上涨, 这里主要是指2021年的第一季度, 当然, 2020年手机的销量整体暴跌过, 但 2020 年pc的销量没有暴跌, 原因可能是居家远程办公增加所致。

3. to-do list 应用市场主要功能

目前市场中 todo list 软件的主要功能包括:
A) 记录待办任务, 跟踪任务完成情况。
B) 番茄钟提醒用户休息, 以提高工资工作效率, 这个大部分的软件都有, 特别是中国国内的软件应用。
C) 监控电脑操作行为, 记录时间浪费在哪里, 比如 toggl。
D) 提醒用户, 建立良好的习惯， 这个还比较少见。
E) 团队合作功能, 这个 todoist 在提供。
F) 跨平台, android, ios, linux, windows, mac 都要提供。

当然还有其他功能, 但主要的功能只有上面这些。

4. to-do list 应用主要竞争格局

国外占头部位置的有:
todoist, any.do, microsoft to do, google task 等等。
从 google play 中查询到的数据, todoist, any.do, microsoft to do, 累计下载量都超过1000万, 其中 todoist 在官网博客中写明用户量为 500 万, any.do 声明, 到 2020年为止, 用户量为 3000 万, 从 getlatka 查询到的数据, todoist 的年收入为 $1100万, any.do 的收入为 $1000 万, todoist 的用户比 any.do 少但收入还高. 而 microsoft to do 免费, 所以没有收入。

中国国内， 占头部的有:
滴答清单, 专注清单, 极简待办,番茄土豆, 指尖时光, 时光序, 小日常等, 从禅大师网站查询到的数据, 这些应用的下载量都达到每天几百上千次的下载量, 有些甚至高达4000次/天.
