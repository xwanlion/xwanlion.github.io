咨询   2022-02-19


工作经历:

第一份软件开发工作

我工作已经有20年, 前面10年是在工厂车间度过, 2012年12月份才开始进入 IT 行业, 第一份工作是 vb6, 在这期间, 我开发过一个系统, 不是很成功, 原因是系统有3个模块， 包括基础资料管理, 业务流程控制和财务管理, 基础资料和业务流程控制大概用2个月的时间开发完成, 但是财务模块, 没有达到甲方的要求, 最后被迫做精简化处理, 并且超时了, 当然， 最后客户还是同意支付合同款, 金额应该有所折扣, 具体不清楚。 一年多后, 也就是14年3月份我就离职, 原因有二:

  1. 我觉得系统开发过程中, 需求没有明确就开始着手编码， 特别是财务模块, 这个有问题, 从我的角度来看, 当时我没有需求分析和设计的经验, 所以我觉得应该有人帮设计出原型后再让我编码比较合适, 公司没有找人设计, 责任不在我这边， 再加上开发过程中产生的压力和气愤, 所以想离职。
  2. vb6 已经过时, 我必须再次学习其他语言.

自学 java

离职后, 我自己呆在家里两年, 学 java, 模仿上家公司的系统用 java 开发出来, 框架是 hibernate，spring， struts， ms sqlserver， 前端用的是 easyui, 两年后完成了, 存在这大量的bug, 但我已经没有毅力在坚持了, 有过把系统拿出去卖的念头, 但是已经感到乏力, 没有心思再去坚持了, 所以想去找 java 方面的工作了.

找到 java 的工作了

我自己购买了阿里的服务器, 把开发出来的系统部署上去, 然后拿着这个系统去面试, 终于, 找到工作了, 用了20天的时间。

找到工作后异常兴奋, 觉得很有盼头.

公司做的是司法厅业务, 当时加班的时候我都是很高兴, 周末别人休息了, 我自己去上班也是很高兴, 下班的时候依然有身轻如燕的感觉, 那时候状态良好。 不过好景不长, 由于要驻场到司法部门, 那里的管理严格, 而且有些行为我不赞同, 比如突击动作, 下面举例子:
当时公司接了一个系统, 用来调查司法部门有多少人员, 多少软件或者设备之类的, 从需求分析到设计到编码, 上线, 以及统计数据的收集, 只需 15 天, 整个组投入 7 - 8 个人, 每个人开发一两个小的模块, 没有测试人员参与, 上线维护的时候是我来维护, 每个模块都有 bug, 统计数据无法显示出来, 司法厅的人就让我现场写 sql 语句来统计, 很痛苦, 更要命的是, 上线前一天下午, 他明明知道还有bug没有处理完成, 却依然要求加入新的模块, 最终那个要求没有实现, 统计数据收集完毕后就停用系统了。

像这种不合理的现象很多, 我看着很不舒服, 再加上我的基础薄弱, 修复问题每天只能处理两个 bug, 技术总监天天问我今天处理了多少个问题, 我年龄比其他人大, 自己都感觉不好意思, 记得有一次, 准备下班的时候, 有个同事在办公室里面大声喊道: 有个家伙每天只能修复两个 bug, 但还需要天天加班, 说完后其他人哄堂大笑。

从那以后, 我就再也打不起精神了, 我觉得这样的环境好像是在摧残人, 不是在锻炼人, 您要说这些能不能让人成长? 我自己的答案是, 当你还没有掌握基本的东西的时候, 可以提高你的技术水平, 但是想要深入, 很难, 深入需要静下心来, 可这样的环境无法静下心来。

不到一年的时间,我递交了辞职信, 离职的时候, 所有代码都交接清楚了， 我以为再到人事部门签字后就可以了, 没想到, 人事主管跟我说, 老总让我到他办公室一趟。

我内心有愧疚, 感觉是他力排众议招我进去的, 所以我很不情愿过去, 但还是走进了他的办公室, 进去的时候他正在打电话, 很久, 应该有 20 分钟, 我坐在那里, 感觉度日如年, 打完电话后, 我不知道说些什么, 等着他开口, 没想到他也没有开口, 很久才问我， 为什么要离职？ 我跟他说, 我适应不了, 他回应说, 你不是自己可以开发一个系统吗？ 怎么不行呢？ 我说, 我自己开发的时候, 速度慢没关系, 但是这里不行。 然后我就开始抱怨, 他语重心长的说, 客户信任我们才把问题交给我们, 我没有回答这个问题, 我就这样离开了那家公司！

第二份 java 工作

离开后, 我以为我再也无法找到开发方面的工作了， 但很幸运, 我又找到了！

这是一家以货柜托运业务为主的公司, 本来他们想要建成一个托运平台, 让工厂在上面下单, 司机接单, 以此来运营, 但很遗憾, 项目失败, 原因是, 货柜里面的货物价值几十万甚至上百万, 轻易更换合作供应商让工厂感到担忧, 所以没有工厂来下单, 公司只好转变运营方式, 把自己当作一个车队来做, 自己去找工厂拉订单, 然后再找司机来托运, 但这个问题公司作为第三方插足进去, 能够赚的利润就少了, 所以就经常出现司机接了单, 但是任务往往延迟完成, 自己先去完成其他公司的再说。 所以, 后来, 公司自己买大卡车, 雇佣司机来开车, 但是又出现别的问题， 就是司机开车赚的少, 所以就开车去拉别家公司的货柜, 在加上这个时候赶上贸易战, 公司后来又着手把大卡车处理掉了。

这个时候我也想离职了, 原因是我觉得贸易战后, 外贸行业肯定会遭受打击, 程度有多严重我当时不清楚, 但我觉得这个行业多少都会受到影响, 在加上我想接触到高并发方面的业务， 所以就提交了辞职信。

这家公司的老总对我很好, 买大卡车的时候, 他想说服我入股, 然后赚利润, 但由于我没有合伙入股的经验, 以及贸易战的因素, 我拒绝了。 唯一不爽的地方就是, 申请高新技术企业的时候, 他让外面专门提供高新技术企业认证的公司来帮助, 写技术说明的时候， 这些人拼命的要求我一定要往高大上的技术靠拢, 我一看, 高大上的技术术语就那么几个, 提供认证的系统数量就已经有10个了, 我怎么靠拢? 我总不能像写小说那样发挥自己的想象力吧, 再说了, 这10个系统也是没有10个系统的, 是把一个系统的每个模块当作一个独立的系统来写的, 我觉得这样做迟早某一天会出问题的, 写着写着, 我就窝火了, 最后拒绝签字。

到现在为止, 我用到的技术包括: struts, ibatis, spring, spring mvc, mysql, vue.js.

尝试做自己的项目

出来后, 已经是 19 年中旬了, 我感觉良好, 我觉得下一份工作应该如我预期的那样, 会更好, 于是先玩了半年, 打算在这半年的时间里面, 做一点自己想做的东西,半年之后在决定是否要找工作。 我想做一个网站, 去购买别人刚创作出来的主旋律, 然后通过比赛的方式来编曲, 在通过比赛的方式来决定谁可以获取歌唱这首歌的版权, 以此来一步步推火一首歌曲, 大概用了一个月多月的时候, 项目上线了, 我开始去拉人, 我拉了两个月, 在 QQ 的音乐群里面拉人, 有两个人同意过来注册, 但最后都没有过来。

这个时候我叹气了, 我感到人际交往方面的缺点暴露无遗, 似乎这条路我走不通, 不过这个时候已经赶上了肺炎, 也只好呆在屋里了, 但我不能干等着啊, 我总得做一点东西啊, 正好, 我想到了要学 android， 所以就开发了一些项目来练手。 到 20 年 7 月份的时候, 我必须要找工作了, 但这个时候发现要求更严格了, 我的年龄偏大, 而且使用的技术偏老, 最新的 spring boot, spring clound 这些我都没有使用, spring 家族我只是接触了 spring mvc, 当然还有公司愿意要我, 但要么是技术太老我不愿意去, 要么就是软硬件结合要求我学c, 而且工资要减少很多, 所以我都不愿意去.

这家公司宣称自己是做直播的

最终, 我把目光转到了老家的省会城市, 找到了, 他说是做直播的, 我一看又兴奋了, 我觉得在这个城市里面, 回家虽然有点距离, 但总比一线城市好得多吧, 再加上肺炎来了以后, 直播正在火着呢, 我觉得机会又来了, 所以又天天加班, 周末也自动去加班。

这个时候有一个奇怪的问题我无法理解, 这是一家新成立的公司, 在我进去之前, 已经有一个前端的技术人员进去了, 他每天上班的时候, 总是迟到, 基本上都是10点多才到, 最严重的一次是, 有天早上下了一点雨, 雷声大, 但是雨量不大, 他竟然到了中午12点半了以后才到来, 那时候公司包吃, 送午饭的时候已经送过来了, 但是他还没有到, 他来了以后才吃午饭, 我提醒他说, 兄弟, 你做得过分了, 自从那以后, 他提早了一点点, 但还是要到10点钟才来上班.

之后又招来了一些同事, 由于我这个人比较内向, 不爱说话, 他们几个人合得很来, 我就显得有点孤立, 当然出去吃午饭的时候, 他们有时候也会叫上我. 但我感觉只有我一个人在努力, 我依然觉得他们不把这个工作当作一回事, 有的同事上班的时候带着耳机, 在那里摇头晃脑, 老总曾私下里跟我说让我来做管理, 但由于没有公开宣布, 我觉得我没有权利去说他, 所以就当作没看见, 有一次正好被老总看到了, 他有点不好意思, 但随后, 他又当着老总的面带上了耳机, 头又开始摇起来。

公司并没有对这些同事进行任何处理, 哪怕是单纯的警告都没有, 我心里面就开始窝火了. 带耳机摇头的那位同事是 php 开发的, 因为直播系统的后台是 php, 之前外包给别人所以招他进来负责， 我是 java, 上班总是迟到的那位则是前端的, 由于 php 开发的同事技术比我好, 经验也比我丰富, 所以公司决定让他来管理所有技术上的问题, 老总私下里跟我说, 我同意了, 但是做 php 的这位同事对我有点意见, 原因是我气过头了以后得罪了他.

所有的人开始积极起来, 这个时候开始开会讨论项目的问题了, 讨论的是电商系统, 用 java 开发的, 我来负责后台, 原型设计老总来负责, 直播系统外包给别人了, 所以不用考虑了。 开会的时候, 我发现， 老总总是心不在焉, 我不知道为啥, 项目讨论一次, 我的后台就得修改一次, 刚开会了两次, 我后台的代码就感觉很乱了, 我知道原型设计不全, 曾建议老总找找个产品经理来负责, 可是他不同意, 执意要自己来负责。

我觉得这样做不行, 前端的同事在等着我把文档以及接口开发出来, 可我的工作无法展开, 因为原型搞总是出问题, 每次开会都是大的变动, 而做管理的这位同事, 积极性虽然是有点, 但事情似乎是跟他没有关系, 他也没有要着急的意思。 我觉得只有我一个人在努力, 最后心想, 要么全部使用 php 来开发, 让做管理的这位同事派上用场, 要么是找技术好一点的人进来负责 java 这边的, 公司分成两个组并行开发, 反馈到老总了以后, 老总同意晚上过来讨论这个问题, 但我心里面已经决定要离开了, 因为我觉得跟我没有任何关系了, 即便是另外招人进来负责 java 这边的, 我也觉得公司的这个文化氛围, 不会有什么好的结果, 况且这些同事已经形成合力, 明里暗里给我施压, 这样下去的话, 即便有结果, 我的付出会比别人更多, 而且不止一两倍。

所以, 晚上开会的时候, 我极力让老总用 php 来开发, 放弃 java, 但老总决定再另外招人进来, 我委屈, 眼泪禁不住流了出来, 其他同事用开玩笑的语气试图逗我开心, 但我开心不起来, 我看到他们脸上露出一丝丝苦笑, 很尴尬的样子。 我提前离开了会议, 回到住处后, 接到老总的电话, 他跟我说, 明天周六, 但前端的同事想跟我一起加班, 把项目中的一些 bug 处理掉, 我猜测这只是一种借口, 他应该是有什么话要跟我说, 所以我答应了。

第二天早上我去的很晚, 过去我什么都没有做, 只是坐在那里等着他开口说话, 中午了, 他依然没有开口, 我就起身去吃午饭, 他跟着我出来, 一路上一句话都没有说, 不过我们很默契的走进了一家餐馆, 各自点了一份, 吃到一半的时候, 他跟我说, 公司只是想申请高薪技术企业才开发项目而已。

我一句话都没有说, 吃完后就回到办公室, 把自己的东西收拾好, 离开了办公室。

晚上的时候, 我在微信上把另外一个老总骂了, 他只是名义上的老总而已, 不是公司的实际控制人, 前面提到的老总是实际控制人, 这个我们在公司内部已经知道, 但他辩解说公司是有心做项目的, 前端的同事说得不对。 我跟他说, 为什么招人进来了, 却没有交接代码过来, 让 php 的同事干等, 为什么同事上班做得这么离谱, 却没有受到处罚？ 只有他的话才能解释所有这些离奇古怪的事情。

接下来的几天, 公司搬迁, 换另外一个办公地址, 其他同事都去帮忙了, 我赌气, 没有去, 等于缺席矿工了差不多一周的时间。

然后我去上班了, 办公室里面的气氛死气沉沉, 我知道是我引起, 我知道我自己像个小男孩在撒脾气, 而且其他人对我无能为力。 这些天, 前端的这位同事天天跟着我出去吃午饭, 但没有太多的话, 他提醒我说, 不应该这样说话, 我跟他说, 我有一种被骗的感觉。

本来公司还想让我把电商的后台管理系统做出来, 我自己就着手去做原型设计, 我跟老总（实际控制人）说, 我的原型画得不怎么样, 但我觉得逻辑和流程上应该没有什么大的问题, 他同意了. 大概过了三天时间, 公司开会, 决定放弃 java， 全部使用 php 来开发， 而且不是做电商系统, 而是另外开发临时工作发布平台, 让工地上找临时工的人有一个招工的地方。 我觉得没有我什么事情了, 于是提交了辞职信, 但公司不同意, 觉得我可以留下来做 android 方面的开发, 但我去意已决, 坚持要离开, 为了找到合理的理由, 我从网上找来招工平台 app 使用率的调查报告, 跟他们说, 大部分的人找工, 使用 app 的频率都不是很高, 都是通过浏览器来找的.

最后, 我离开了, 离开的前一天, 两位老总（实际控制人和名义上的法人代表）都在办公室里面, 什么话都没有说, 其中有一个人眼睛很红， 精神状态不是很好。 大约一周后, 公司的实际控制人打电话给我, 想跟我见个面, 我去了, 他问了一些问题, 我记得当时跟他说, php 开发的这位同事, 技术上没有什么问题, 关键的问题是招工的时候是我面试了他, 对他来讲, 我是个老资格, 可能我性格以及能力上有问题, 但他不好意思说出来, 结果问题一直拖着, 越拖越大, 您就给他两三个月的时间, 看他招工系统做得怎么样, 如果连个样子都没有出来, 那就解散了吧, 另外组建团队, 要不然这个氛围传下去, 那肯定不行。

打官司

到这一步为止, 我已经没有心思去找工了, 我感觉我无法应付办公室里面的这种氛围, 这种氛围让我心力疲惫, 所以又想着自己去做一些东西。 但这个时候, 公司还有一个多月的工资拖着不发给我, 临近21年春节前, 我催了他们几次, 过年之后我有催了几次, 但还是不发, 他们也同意要发给我, 只是说没有钱, 但我发现他们并没把未发工资部分统计出来给我, 我根本就不知道还有多少钱没有发放。 催了几次之后, 已经过去6个月了, 我就投诉到了劳动监察大队, 公司同意在 2021年5月底发, 到了 5 月 31 日晚上, 我在微信里面发了一个问号, 回复说再等三天, 但这个时候我已经不相信他们了, 第二天儿童节, 我到劳动仲裁委那边去了, 现在, 已经进入了强制执行阶段, 但公司账户上没有一分钱. 这期间我到过公司办公室, 也见到了名义上的那位老总, 人长胖了许多, 但依然说没有钱, 我直接把话说穿了。
现在, 公司还拖欠着我的工资.

开发象棋软件， 人生拍卖会软件, 待办应用软件

我决定要打这个官司, 一方面我觉得自己太天真了, 应该增加点阅历, 让自己的人生丰富起来, 这些事情自己不可能一直忍者, 但是不想忍的话, 打官司那是注定要发生的事情, 所以我觉得应该自己去打官司, 不请律师, 另一方面应该让公司领导层知道, 这个世界, 所有行为都是有代价的。

但打官司之余我能做什么呢? 一开始是想做象棋软件, 市场上的象棋软件, 有4个细分市场: 在线对弈, 人机对战, 训练下棋能力, 还有干脆封装成 jar 包让别人去做, 我发现即便这样, 还是有人采用传统的方式来下象棋, 一般都是中老年人居多, 我觉得是因为这种方式能够增加感情, 所以还有人采用这种方式, 所以我采用 b/s 模式, 把服务器装在手机端, 做成一个 app, 让他们用浏览器通过wifi网络来下棋, 这样就不用再去购买棋盘了, 再加上手机有热点wifi, 到哪里都可以下象棋, 何乐而不为呢？

等到我编码完成想去推广的时候, 发现有问题, 象棋市场都是免费的, 我这个用户群体只能去地推啊, 这个成本是很高的, 可是我的象棋软件无法收费啊, 别人都免费我怎么收费？ 所以放弃了, 现在有 bug 我也不维护了。

接着我想做一个人生拍卖会活动的 app
, 采用的技术跟象棋软件一样, 唯一不同的是, 象棋软件的前端采用原生 html/js 的方式, 人生拍卖会的这个软件, 前端使用 vue.js 来做。 在刚开始做调查的时候, 我也知道用户量很小, google 关键词分析发现, 每天只有10-13个人搜索, 由于自己没有做推广方面的经验, 不知道这个量意味着什么, 心理就想, 用户量小没有关系, 没有竞争, 我开发了以后收费, 对于这个用户量, 我如果能够收每人1美金, 也是可以维持温饱了, 所以就做了, 花了1千多块钱找美工, 然后又花了几百块钱找 app 评审网站去上传, 然后又花了几百块钱到 google 那里投广告, 但到现在为止, 依然没有人下载, 个人估计, 下载的1-2个用户应该是评审网站的人。

到这这一步为止, 我就想, 前面的这些, 都是我想去做别人需要的东西, 但是我不知道别人怎么想啊, 不如这样, 我自己开发一个我需要用的东西, 所以就用 flutter 开发一个待办事项应用程序, 类似于 todoist。 开发出来了, 我还想继续优化, 继续完善, 但我已经没有钱了, 我只能找工作去了。

找不到工作了

可是, 我已经找不到工作了, 到 boss 上面投简历, 人事部门和经理级别的人非常高兴, 但是简历递交到技术人员手里, 往往被刷掉了。

我所接触到的技术中, 有 vb6, java, hibernate, spring mvc, spring boot, struts, mybatis, jsp, freemaker, html/js/css, extjs, easyui, vue.js, elementui, websocket, mysql, oracle, sql server, sqlite, android, dart/flutter, 我觉得我能胜任工作, 但是面试官说不行, 我所使用的技术框架太老了, 我达不到他们的要求。

我现在去面试前, 都不再刷题了, 我知道我应该刷题, 但是已经没有心思了。

对开发的一些感觉

  • N多天不写代码也不会觉得怎么样
  • 当有写代码的冲动了以后, 最多持续2-3天, 很少到1周
  • 写代码总觉的自己很慢
  • 自己也知道应该多读技术书籍, 但是看着看着就没有兴趣了, 往往被社会学方面的书籍吸引了
  • 写代码总会出现很多 bug, 找到原因后, 往往都是自己犯的低级错误引起

我的问题

  • 我还能找到开发方面的工作吗, 还是应该转行?
  • 如果还做开发, 我应该向后端发展呢？ 还是前端比较好？
  • 现在的混合开发 flutter 非常火, 这值得去研究吗？ 会不会像 struts 一样, 过几年后又被淘汰掉呢？
  • 我业余写了一些歌曲主旋律, 觉得要是有人拿去演唱, 这对我的帮助会非常大, 主要是带来名气, 然后我可以通过名气来宣传我的软件, 这样做可行吗？ 我不清楚音乐行业的人会对这些旋律作出什么样的评价, 我自己曾经出钱让别人帮忙编曲, 但是我觉得他编曲不好, 可能是我给的钱太少了吧？ 因为编曲可以使用虚拟音色也可以使用真人演奏, 效果不同的， 而且要推出完整的歌曲大概需要10万元人民币, 我没有钱啊, 所以我希望别人拿去用, 我顺带分享版权费用, 但是另一方面, 我试图去联系抖音上的网红, 他们都不回应, 即使回应, 也不会留联系方式, 我看到网上说, 你要么有对等的名气, 要么出对等的价钱, 否则你不要找名人, 大家都不傻。
  • 我40岁了， 还没有结婚, 性格上变得孤僻了, 而且越来越偏激了, 该怎么办？即便要转行, 我也还需要靠开发工作来维持生计, 有临时解决办法吗？
  • 如果转行, 以我的性格可以转什么行业？ 我不想丢掉多年来积累下来的经验, 如果掉丢了, 我的代价太大了
  • 以您的经历, 想让您给我个评价, 是哪方面出现了问题, 是学习方法吗？ 还是性格？或者其他方面出现了问题？ 我觉得性格的居多。
  • 前段时间我在家乡的省会城市找到了工作, 做前端的, 但是二次面试的时候, 他们放飞机了, 让我白跑一趟, 所以我觉得他们招用我可能是勉勉强强的事情, 在加上我想找更高工资的工作, 所以放弃后跑到一线城市来了, 但是现在发现, 一线城市要求更加严格, 似乎一线城市的竞争更激烈? 我的感觉是不是对的？
